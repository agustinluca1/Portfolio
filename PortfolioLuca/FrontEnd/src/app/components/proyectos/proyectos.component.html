  • Portfolio Web
    • Fecha de realización: 5/2022
    Descripción: El proyecto se basó en realizar mi propio portfolio web. Tanto el diseño del front-end como la lógica del back-end.
    Para el front, utilicé HTML, CSS, Angular, algo de TypeScript y la librería NgxChart.
    Para el back, un poco de Java y SpringBoot.
    Sé que tiene muchas cosas a mejorar, sobre todo para que el proyecto sea responsive (quizás porque me quedaba sin tiempo para la entrega).
    En el gráfico tambien tengo muchos detalles, no estoy para nada conforme como quedó, no pude agrandar el tamaño de cada dona para que se lea la habilidad completa, ni hacer que me respete el contenedor cuando se reduce el tamaño de la pantalla. Otra cosa que me hubiera gustado y no pude lograr, es que la animación del gráfico comience cuando lo veas y no apenas carga la página.
    Bueno y así hay un montón de detalles que me gustaría pulir. Se podría decir que este es el primer prototipo de mi portfolio.
